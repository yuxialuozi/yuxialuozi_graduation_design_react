# 项目初始化提示词

请帮我创建一个完整的 React + TypeScript 前端项目，项目名称为 `yuxialuozi_graduation_design_react`。

## 项目背景

这是一个**租户合同交易与房租状态管理平台**的毕业设计项目，核心功能包括：
1. 租户管理（租户信息录入、信用评估）
2. 合同管理（合同全生命周期管理：草拟、审批、生效、执行、变更、续签、终止）
3. 房源管理（房源信息、状态管理）
4. 财务管理（租金账单、在线支付、对账）
5. 预警系统（租金逾期预警、合同到期提醒、房屋空置预警）
6. 数据可视化（仪表盘、统计报表）

## 技术栈要求

### 核心技术
- **框架**: React 18+
- **语言**: TypeScript
- **构建工具**: Vite
- **UI组件库**: Ant Design (antd) 5.x
- **图标**: @ant-design/icons
- **路由**: React Router v6
- **状态管理**: Zustand 或 Redux Toolkit（推荐 Zustand，更轻量）
- **HTTP客户端**: Axios
- **图表库**: ECharts for React (echarts-for-react)
- **日期处理**: dayjs
- **样式**: CSS Modules + Less/Sass

### 开发工具
- ESLint + Prettier（代码规范）
- TypeScript 严格模式
- Git hooks（可选）

### 3. 配置 Vite (vite.config.ts)
```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import path from 'path'

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
  server: {
    port: 3000,
    open: true,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/api/, ''),
      },
    },
  },
  css: {
    preprocessorOptions: {
      less: {
        javascriptEnabled: true,
        modifyVars: {
          // Ant Design 主题定制
          '@primary-color': '#1890ff',
        },
      },
    },
  },
})
```

### 4. 主要功能页面需要实现

#### 登录页面 (src/pages/Login/index.tsx)
- 使用 Ant Design 的 Form 和 Input 组件
- 表单验证
- 记住密码功能
- 美观的渐变背景

#### 仪表盘 (src/pages/Dashboard/index.tsx)
- 统计卡片（租户总数、合同总数、本月收入、待处理事项）
- 图表展示（收入趋势图、合同状态饼图、房源状态柱状图）
- 最近操作列表
- 快捷入口

#### 租户列表 (src/pages/Tenant/List/index.tsx)
- 搜索表单（姓名、手机号、状态筛选）
- 数据表格（支持分页、排序）
- 新增/编辑/删除操作
- 查看详情

#### 合同列表 (src/pages/Contract/List/index.tsx)
- 合同状态筛选（待生效、执行中、已完成、已终止）
- 合同时间筛选
- 表格展示合同信息
- 合同状态流转（状态机展示）
- 合同详情查看
